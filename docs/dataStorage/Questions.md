- 字符集和编码方案的区别
字符集是字符列表，而编码方案是它们以二进制形式表示。
Unicode是字符列表
UTF-8、UTF-16 和 UTF-32 使用 Unicode 字符集，但对字符的编码不同

-  我的文件在 Ascii 中， 但它使用 Utf - 8 解码器解码正常。为什么？
因为 UTF-8 向后与 ASCII 兼容。

- 如何通过网络将字母 A 等传送到另一台计算机？
若要将文本存储为二进制数据，必须为该文本指定编码。
数据传输也是一样的,必须指定编码
